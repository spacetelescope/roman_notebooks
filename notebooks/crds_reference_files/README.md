# CRDS Reference Files

In this direcotry you will find a collection of notebooks which guides users through understanding the structure, content, and usage of Calibration Reference Data System (CRDS) reference files for the Roman Wide Field Instrument (WFI).

The notebooks in this workflow are designed as standalone tutorials but can be followed sequentially for a complete introduction to how reference files are
retrieved, inspected, and visualized.


- crds_reference_files.ipynb: is a general overview about CRDS. With this notebook you learn what CRDS is, how reference files are matched and delivered, and how to use `crds.getreferences()` and `crds.getrecommendations()`. 


### Individual Reference File Notebooks

In this directory you will find notebooks for most of the reference files used by the Roman Calibration Pipeline. The list of notebooks is:

- bad_pixels_mask_reffile.ipynb  or MASK reference file
- saturation_reffile.ipynbdark_reffile.ipynb or DRK reference files
- saturation_reffile.ipynb of SATURATION reference file
- reference_pixel_reffile.ipynb or REFPIX reference file
- distortion_reffile.ipynb or DISTORTION reference file
- photom_reffile.ipynb or PHOTOM reference file
- gain_reffile.ipynb or GAIN reference file
- readnoise_reffile.ipynb or READNOISE reference file
- area_reffile.ipyn or AREA reference file
- psf_reffile.ipynb or PSF/ePSF reference file
- linearity_reffile.ipynb or LINEARITY reference file
